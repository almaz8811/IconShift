# IconShift

Windows-приложение для просмотра и переноса файлов/ярлыков между рабочими столами пользователей.

## Возможности

- Три панели с содержимым Desktop (крупные иконки)
- Выбор профиля пользователя или «Общая папка» (Common Desktop) в каждой панели
- Drag & Drop между панелями (перемещение; с Ctrl — копирование)
- Приём файлов из Проводника
- Двойной щелчок — открыть файл/папку
- Кнопка 📁 — открыть текущую папку в Проводнике
- Автообновление списков при изменениях на диске
- Один экземпляр приложения (mutex)

## Требования

- .NET 8.0 SDK
- Windows 10/11

## Сборка и запуск

```bash
cd IconShift
dotnet restore
dotnet build
dotnet run
```

Готовый exe: `bin/Release/net8.0-windows/IconShift.exe` (после `dotnet build -c Release`).

## Использование

1. Запустите программу.
2. В каждой панели выберите пользователя или «Общая папка».
3. Перетаскивайте файлы и ярлыки между панелями.
4. Дважды щёлкните элемент, чтобы открыть его.
5. Кнопка 📁 открывает выбранный Desktop в Проводнике.
