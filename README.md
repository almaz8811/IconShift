# IconShift

Программа IconShift - это приложение для просмотра ярлыков программ с рабочего стола Windows.

## Возможности

- Отображение всех ярлыков с рабочего стола
- Просмотр названия и пути к программе
- Дважды кликните на ярлык для запуска программы
- Кнопка "Обновить" для обновления списка ярлыков

## Требования

- .NET 8.0 SDK
- Windows 10/11

## Сборка и запуск

```bash
# Войдите в директорию проекта
cd IconShift

# Восстановите зависимости (если требуется)
dotnet restore

# Соберите проект
dotnet build

# Запустите приложение
dotnet run
```

## Структура проекта

```
IconShift/
├── IconShift.csproj       # Файл проекта
├── Program.cs            # Точка входа в приложение
├── MainForm.cs           # Основное окно приложения
└── README.md             # Документация
```

## Использование

1. Запустите программу
2. Список ярлыков с рабочего стола будет автоматически загружен
3. Дважды кликните на любой ярлык для запуска программы
4. Нажмите кнопку "Обновить", чтобы обновить список ярлыков
